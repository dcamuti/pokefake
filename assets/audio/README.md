# Audio

Cartella predisposta: il gioco carica automaticamente i file elencati qui sotto
**se presenti**; in loro assenza usa i beep chiptune integrati (per gli effetti)
o resta in silenzio (per la musica). Nessuna modifica al codice richiesta.

## music/  (loop automatico, volume 40%)
| File | Quando suona |
|---|---|
| `title.mp3` | Schermata del titolo |
| `overworld.mp3` | Esplorazione della mappa |
| `battle.mp3` | Battaglie (selvatiche e allenatori) |

## sfx/  (opzionali, sostituiscono i beep)
`menu.mp3, a.mp3, b.mp3, hit.mp3, super.mp3, heal.mp3, lvl.mp3, catch.mp3,
faint.mp3, run.mp3, badge.mp3`

Nota: la musica parte al primo tasto premuto (i browser bloccano l'audio
prima di un'interazione dell'utente). Tasto M = audio on/off.
